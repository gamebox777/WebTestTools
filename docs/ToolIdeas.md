# 今後の追加機能・ツールアイデア一覧

WebTestToolsのさらなる利便性向上のため、今後追加を検討・予定している便利なツールのアイデアリストです。

## 1. 開発・デバッグ向け（Developer / Debug Tools）

*   **JSON フォーマッター ＆ バリデーター**
    *   **概要:** 1行に圧縮されたJSONを見やすく整形（Pretty Print）したり、構文エラーがないかチェック（Validate）する機能。テストや仕様書作成時にあると便利。
*   **エンコード / デコード ツール**
    *   **概要:** URLエンコード / デコード、および Base64エンコード / デコード機能。APIパラメータの検証や、Basic認証用文字列の生成などに使用。
*   **Unixタイムスタンプ 変換ツール**
    *   **概要:** タイムスタンプ（数値）を日時に、または特定の日時をタイムスタンプに変換する機能。DBの検証などに最適。
*   **テキスト差分チェッカー (Diff Checker)**
    *   **概要:** 2つのテキストを比較し、変更行や文字をハイライトする機能。設定ファイルの差分確認などに重宝。
*   **正規表現テスター (Regex Tester)**
    *   **概要:** 正規表現のパターンと対象文字列を入力し、マッチ状況をリアルタイムハイライトする機能。バリデーションの実装テストに。
*   **JWT (JSON Web Token) デコーダー**
    *   **概要:** JWT文字列を貼り付けてヘッダーやペイロード（中身）をデコード表示する機能。API認証のトークン解析に。

## 2. データ作成・変換向け（Mock / Converter Tools）

*   **ダミーデータ生成（Mock Data Generator）**
    *   **概要:** 「氏名」「メールアドレス」「電話番号」などのダミーユーザー情報をランダムに大量生成し、JSONやCSVで出力する機能。
*   **パスワード / ハッシュジェネレーター**
    *   **概要:** ランダムなパスワード生成、および文字列のハッシュ化（MD5, SHA-256など）。テスト用アカウント作成等に。
*   **QRコード生成ツール**
    *   **概要:** URLなどを瞬時にQRコード画像に変換。スマホ実機での検証に便利。
*   **UUID / GUID ジェネレーター**
    *   **概要:** 一意なIDの一括生成機能。DBのテストデータなどに利用。
*   **HTML エンティティ変換 (Entity Encoder/Decoder)**
    *   **概要:** `<` や `>` などを `&lt;` や `&gt;` に変換/復元。XSSテストやスニペット作成に。
*   **SQL フォーマッター**
    *   **概要:** 1行のSQL文をきれいにインデントして整形する機能。ログからの長大クエリの解読に。

## 3. デザイン・フロントエンド向け（Designer / Frontend Tools）

*   ~~**アスペクト比計算機 (Aspect Ratio Calculator)**~~ → *実装済み*
    *   **概要:** 画像サイズの計算や、正確な比率の算出。
*   **カラーコントラストチェッカー**
    *   **概要:** 背景色と文字色から、WCAG基準などを元に文字の視認性を判定するアクセシビリティツール。
*   **CSS ジェネレーター (Box-Shadow / Gradient)**
    *   **概要:** 影やグラデーションをGUIでプレビューし、結果のCSSコードを生成する機能。
*   **キーボードイベント・キーコード取得 (Keycode Viewer)**
    *   **概要:** 押したキーの `keyCode`, `key`, `code` 情報を画面に表示するツール。ショートカット実装等に。
*   **デバイス画面サイズ・リファレンス**
    *   **概要:** スマホやタブレットの代表的な解像度（ピクセル数・比率）の一覧チートシート。レスポンシブ対応の確認に。

## 4. その他（Misc）

*   **Cron式 ジェネレーター**
    *   **概要:** 「5分おき」「毎日9時」などのUI選択からCron式（例: `0 9 * * *`）を生成する機能。バッチ処理などの設定忘れ防止に。
