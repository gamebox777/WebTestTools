# Test Tools Portal (テストツールポータル)

Webシステムの検証や動作確認をサポートするための、便利なツール群をまとめたポータルサイトです。

## 概要

このプロジェクトは、フロントエンド開発、バックエンド検証、デザインアライメントなどのさまざまなテスト状況で必要となる各種アセットやデータを、ブラウザ上から簡単に生成・取得することを目的としています。
現在、以下のツールが利用可能です。

---

## 🛠 利用可能なツール

### 1. テスト画像生成ツール
Webサイトのレイアウト検証やアップロード機能のテスト用に、指定した設定でダミーファイルを一括生成してダウンロードできるツールです。

- **対応フォーマット**: PNG, JPG, PDF, XLSX (Excel)
- **指定可能なパラメータ**:
  1. 生成するファイルの枚数
  2. 画像サイズ (幅・高さ) ※PDF/Excel以外
- **生成内容**: 中央に「test_1.png」などのファイル名が描画された、シンプルな背景のファイルを作成します。  
- **ダウンロード形式**: 複数枚生成した場合でも1つのZIPファイルにまとめられて自動的にダウンロードされます。

*(※今後のアップデートにて、テキストカウンタやテキストジェネレータなどの追加を予定しています)*

## 💡 使い方

1. **ポータルトップ** にアクセスします。
2. 利用したいツールのカードにある **「開く」** ボタンをクリックします。
3. ツールの画面（例：画像生成）にて、必要なパラメータ（形式、サイズ、枚数など）を入力します。
4. **「ファイルを生成」** ボタンを押すと処理が実行され、結果がファイルまたはZIPとしてお手元の環境にダウンロードされます。

---

> [!NOTE]
> このツール群自体は、開発者やテスター向けに提供される内部サポート用アプリケーションです。
> バックエンドへのデータ保存（データベース）等は行っておりませんので、生成データや入力内容は保存されません。
